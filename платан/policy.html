Политика обработки персональных данных

1. Общие положения

1.1. Настоящая Политика обработки персональных данных (далее — Политика) разработана в соответствии с требованиями Федерального закона от 27 июля 2006 года № 152-ФЗ "О персональных данных" (далее — Закон) и регулирует вопросы обработки персональных данных, осуществляемой к»компанией Платан-Энерго - подразделение ГК Платан - проектые поставки электронных компонентов для разработки и серийного производства электроники ( далее - Компания ).

Цели обработки персональных данных

Компания обрабатывает персональные данные пользователей только для следующих целей: - Связь с пользователем для консультации и записи на приём к врачу.

Персональные данные

Компания не сохраняет и не хранит персональные данные пользователей после их обработки. Персональные данные, предоставляемые пользователем, используются исключительно для обратной связи и удаляются после завершения общения.

При заполнении формы обратной связи в Клинику пользователь предоставляет следующие данные: - Имя. - Номер телефона. -Email

Правовые основания для обработки

Обработка персональных данных осуществляется на основании согласия пользователя, выраженного при заполнении формы обратной связи.

Пользователь имеет право в любой момент отозвать свое согласие на обработку персональных данных, для этого ему необходимо направить соответствующее уведомление в Компанию.

Порядок обработки персональных данных

Персональные данные обрабатываются в соответствии с Законом, а также с соблюдением принципов добросовестности, законности и прозрачности.

Персональные данные пользователя используются только для указанных выше целей и не передаются третьим лицам, за исключением случаев, предусмотренных законодательством РФ.

После завершения общения персональные данные пользователя удаляются.

Меры по защите персональных данных

Компания принимает необходимые организационные и технические меры для защиты персональных данных от несанкционированного доступа, искажения и уничтожения.

Доступ к персональным данным осуществляется только уполномоченными сотрудниками Компании, которые обязаны соблюдать конфиденциальность информации.

Права пользователей

Пользователь имеет право: - Запрашивать информацию о том, какие его персональные данные обрабатываются Клиникой. - Требовать удаления своих персональных данных, если они были предоставлены.

Изменения в Политике

Компания оставляет за собой право вносить изменения в настоящую Политику. Все изменения будут опубликованы на сайте Компании и вступят в силу с момента их публикации, если иное не предусмотрено условиями.

Контактная информация
